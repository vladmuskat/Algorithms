# 1. Количество слов в тексте

Во входном файле записан текст. Словом считается последовательность непробельных символов, идущих подряд. Слова разделены одним или большим числом пробелов, переводами строк или символами конца строки.

Определите, сколько различных слов содержится в этом тексте.

**Формат ввода**

Вводится текст.

**Формат вывода**

Выведите ответ на задачу.

**Ограничения**

    Ограничение времени 1 с
    Ограничение памяти 64 МБ

## Пример 1

Ввод
```
She sells sea shells on the sea shore;
The shells that she sells are sea shells I'm sure.
So if she sells sea shells on the sea shore,
I'm sure that the shells are sea shore shells.
```

Вывод
```
19
```

## Пример 2

Ввод
```
AA aa Aa aA
```

Вывод
```
4
```

## Пример 3

Ввод
```
a a a a a a a a a a a a a a a a a a a a a a a a a a a a a a a a a a a a a
```

Вывод
```
1
```
